# Playground

A sleek, modern portfolio built with Next.js, TypeScript, and Tailwind CSS. Showcasing projects, blog posts (technical writeups).

## Features

- Minimal and elegant design
- Dark mode support
- Interactive hover preview cards
- Blog with MDX support
- GitHub contributions graph
- Fast and responsive

## Getting Started

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to see it live.

## Tech Stack

- **Framework**: Next.js 16
- **Styling**: Tailwind CSS
- **Language**: TypeScript
- **Animation**: Framer Motion
- **Database**: PostgreSQL (Prisma)
